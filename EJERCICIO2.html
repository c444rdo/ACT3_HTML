Sintaxis del atributo style

La configuración del estilo de un elemento HTML se puede realizar con el atributo style.

El atributo HTML style tiene la siguiente sintaxis:

<elemento style="propiedad: valor;">

La propiedad es una propiedad CSS y el valor es un valor CSS.
